¡Hola! 👋 Soy **Irán Ledezma**, Ingeniero en TIC, Desarrollador Web y un entusiasta de la tecnología con formación en Ciencia de Datos. 

Este repositorio contiene el código fuente de mi portafolio web personal. Fue diseñado desde cero para centralizar mis desarrollos, combinando una interfaz interactiva con un rendimiento óptimo para ofrecer la mejor experiencia de usuario.

## ✨ Características de la Plataforma

* **UI/UX Moderna (Glassmorphism)**: Interfaz que simula un efecto de cristal esmerilado sobre fondos dinámicos y responsivos.
* **Temas Dinámicos**: Soporte nativo y fluido para Modo Claro ☀️ y Modo Oscuro 🌙 gestionado a través de variables CSS globales.
* **Arquitectura Modular**: Construido con el poder de los componentes de Vue 3, manteniendo el código limpio y escalable.
* **Navegación Fluida (SPA)**: Ruteo instantáneo sin recargas de página, impulsado por el motor de Nuxt.
* **Orientado a Resultados**: Secciones diseñadas para conversión, incluyendo descarga directa de CV y formulario de contacto.

## 📂 Investigaciones y Proyectos Exhibidos

Dentro de este portafolio destaco los retos tecnológicos y de innovación en los que he trabajado:

1. **Integración de Biometría Dactilar en Sistemas de Pago Móvil**: 
   Proyecto de investigación enfocado en elevar los estándares de seguridad y mejorar la experiencia del usuario (UX) en transacciones financieras desde dispositivos móviles.
   
2. **Innovación Alimentaria - Tortillas de Champiñones Deshidratados**: 
   Desarrollo de una investigación para crear una alternativa nutricional e innovadora a la tortilla tradicional.

3. **Portafolio Interactivo**: 
   Esta misma plataforma web, desarrollada para demostrar habilidades prácticas en el ecosistema moderno de JavaScript.
